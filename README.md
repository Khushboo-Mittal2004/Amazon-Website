# Amazon Website Clone
This project is a front-end clone of the Amazon website, developed using pure HTML and CSS. The main goal of this project is to practice and strengthen web development skills by recreating a real-world e-commerce interface.

The clone includes key UI components such as the navigation bar, search bar, product sections, banners, and footer, closely matching the design and layout of Amazon. It focuses on responsive design, clean structure, and user-friendly interface to provide a seamless browsing experience across different devices.

Through this project, I gained hands-on experience in layout structuring, styling techniques, Flexbox/Grid usage, and building pixel-perfect designs.

🔹 Features

Responsive homepage layout

Navigation bar with search functionality (UI only)

Product listing sections

Banner and promotional sections

Footer similar to Amazon

Clean and structured HTML & CSS

🔹 Tech Stack

HTML5

CSS3

🔹 Learning Outcome

Improved front-end development skills

Better understanding of responsive design

Hands-on practice with real-world UI cloning
